About DaisyMaths Exercises

DaisyMaths was a maths learning program for ages 3-12 developed by Myles Harding. Around 2003, a web application was introduced that allowed PDF problem sheets to be created from selected exercises. That web application was written in Perl using the mod_perl web framework and Apache FOP to create the PDFs. This site is trying to revive and update that original web application.

The DaisyMaths Exercises web application is written in Java using basic Servlet and JSP technology. It is still very much a work in progress and should be considered to be in alpha state.

The source code is freely available from GitHub. The project provides a number of web applications:

  • An interactive web-form application where problems are answered online.
  • A Google App Engine-ready version of the web-form application.
  • A PDF problem sheet generator application.

Pre-built WAR files, that can be run in a web container such as Tomcat or Jetty, can be obtained here.

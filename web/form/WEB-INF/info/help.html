DaisyMaths Exercises Help

The steps involved in completing an exercise session are:

  1. Choose the exercise.
  2. Configure the exercise on the Settings page.
  3. Answer each problem until the session is complete or you have had enough.

You can click the [info] link next to the exercise name to find out about the nature of the exercise and what its settings mean.

Answering problems

When you have configured the exercise and clicked the Next;> button, you are taken to the Problem page which shows one problem at a time in a web-form. Fill out each blank field with the answer and click the Submit button (or press <Return>) to submit your answer. If correct, you will proceed to the next problem or the Finish page if you have completed all problems in the session. If incorrect, error messages are shown and you can try again.

If the problem has multiple blank fields to fill out (such as Long Multiplication) then the field focus will follow the sequence used to complete the problem. For example, in long multiplication, you fill out the rows from top-to-bottom but each row is filled out from right-to-left (least significant digit to most significant digit). Pressing <Tab> will advance the focus to the next field; pressing <Shift>-<Tab> will back the focus up to the previous field.

If there are multiple fields then typing a number in the current field will automatically advance the focus to the next field so you don't need to use <Tab> to advance manually.

Number pad

At the left of the Problem page is a number pad which allows you to complete problems using the mouse only. Clicking on a number will enter that number in the next field to be filled out (the field doesn't need to have focus).

Clicking the "equals" symbol will submit the problem and clicking the "back-arrow" will move the focus back to the previous field.

Buttons & links

Clicking the Skip button will skip the current problem and show the next problem.

Clicking the Show button will show the complete answer to the current problem: from here you can click the Next> button to proceed to the next problem.

Clicking the Finish button will end the session and show a summary of your results.

Clicking the Restart session link will abandon the results of the current session and start a new session.

Clicking the New session link will take you back to the exercise configuration page so that you can try a different difficulty, etc.

Solution demonstration

Some exercises may include a demonstration of the solution: a Solve button is shown beneath the number pad if a demonstration is available.

Clicking the Solve button will pop up a demonstration panel with Next and X (close) buttons. The panel displays an explanation of each step in solving the problem. Clicking the Next button advances to the next step. When there are no more steps, the Done button is shown.

At any time you can dismiss the demonstration panel by clicking the X (close) button. The problem page itself is disabled while the demonstration panel is open.

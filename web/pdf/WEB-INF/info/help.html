DaisyMaths Problem Sheets Help

The steps involved in making a problem sheet are:

  1. Choose the exercise for which you want to generate problems.
  2. Configure the exercise on the Settings page.
  3. Select the number of problems you would like per problem sheet.
  4. Click one or more of the Download links to receive your PDF.

At any time you can click the [info] link next to the exercise name to find out about the nature of the exercise and what its settings mean.

Downloading

The Download page will present up to three download links:

Download work sheet
Contains the blank problems for the student to solve.
Download answer sheet
Contains the same problems with the answers filled out.
Download work sheet with answers in margin
Has blank problems on the main part of the page with the answers shown in the page margin.

You can download all three PDFs if you wish: they will contain the same problems. If you refresh the Download page — by clicking the Refresh button or using your browser's refresh — then a new set of problems will be generated. This allows you to configure the exercise and choose the number of problems then generate multiple unique work sheets with those settings.
